# Clock-JWS
1. Kode mendukung ESP8266 dan ESP32
2. Terdapat fitur ubah
   - WiFi AP
   - WiFi Router
   - NTP Server atau Manual jam (belum ada DS3231)

   - Reset pengaturan tetapi tidak akan reset jadwal shalat
